Документация UI Kit

Кнопки (Buttons)

Компоненты кнопок для различных действий: основная, второстепенная, неактивная. Поддерживают состояния hover, focus, active, disabled. Имеют три размера.

Поля ввода (Inputs)

Текстовые поля ввода. Поддерживают состояния: normal, focus, error. Возможно добавление иконки.

Ошибка: введите не более 5 символов

Карточки (Cards)

Карточки с заголовком, изображением и текстом.
Имеются модификаторы размера: маленькая, средняя, большая. Карточки могут быть без изображения.
